Reflection and Ideas

The first thing I did before designing the internet artwork was to draw the wireframes and download the images I was going to use for the gallery. I had an idea of designing something like a collage for this internet art, I wanted this design to be simple and attractive. I then resized the images so that they were not going to take time to load, I used seven images so that I don't make the user feel overwhelmed and bored by navigating through the image. For interactivity, I made the images clickable for the user to see and engage with each image separately, when they hover on the images they rotate and become smaller. I made this interactivity because it is interesting, effortless, and simple. The user does not have to apply too much pressure learning how to interact and engage with the content. I used YouTube tutorials as references to help me with the code and ideas, this is the tutorial video I watched to design the internet art, Easy Tutorials.

I would like to do more JavaScript code and learn how to include more JavaScript navigation in my websites. I am planning to watch as may YouTube tutorial videos as I can to improve my coding skills. I feel I have done my best so far to make sure that my websites are interactive and they have meaningful content. I am going to keep on designing more internet artworks to improve this website as I might add it to my portfolio.
